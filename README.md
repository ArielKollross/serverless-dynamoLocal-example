# Tecnologias
 - Node
 - Serverless Framework
 - AWS Lambda
 - AWS DynamoDB

# Rodando a aplicação

Na pasta do projeto, rode no terminal:

sh```yarn```
Instalada as dependências, rodamos dois comandos, cada um em um terminal diferente.

sh```sls offline```

e

sh```sls dynamodb start```
